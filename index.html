Interactive Controls

Mouse - Move to interact with particles

Click + Drag - Push/pull particles

A - Toggle attraction/repulsion mode

1-5 - Change color schemes

R - Reset animation

Space - Pause/resume animation

M - Toggle audio reactive mode

E - Toggle particle emission mode

P - Toggle performance mode

Mouse Wheel - Zoom in/out

Emission Mode

When enabled, click anywhere to create particle fountains!

Color Schemes

1 Twilight (default)

2 Neon

3 Fire

4 Forest

5 Ocean
